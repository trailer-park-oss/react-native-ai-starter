Scaffold Expo apps
with everything wired up.

A CLI that generates production-ready React Native projects with AI built-in. Pick a UI kit, auth provider, AI providers, and theme preset — get a working app with routing, state, and screens.

$ npx create-rn-ai-starter my-app

$ npx create-rn-ai-starter my-app \
    --ui tamagui --auth clerk --preset radix-purple

$ npx create-rn-ai-starter my-app \
    --ai online-openrouter --ai on-device-executorch

What's inside

Every generated project uses these foundations.

Expo SDK + Router

Managed workflow with file-based routing and the full Expo ecosystem.

Zustand

Lightweight state for onboarding, theme, and app state. Persisted onboarding completion.

TanStack Query

Data fetching with caching and background refetch.

React Hook Form + Zod

Type-safe forms with schema validation.

AI Providers

OpenRouter (cloud), ExecuTorch (on-device), and ML Kit with model selector and interactive search.

Radix Design Tokens

12-step color scales, spacing, typography, radius. Light and dark mode. 6 presets.

Provider Pattern

Abstract interfaces for auth, payments, UI, and AI. Swap implementations without touching screens.

Quickstart

1

Scaffold

Run the CLI and pick your options.

npx create-rn-ai-starter my-app
2

Configure

If using Clerk, add your publishable key. For OpenRouter AI, set your API key.

EXPO_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_... OPENROUTER_API_KEY=sk-or-...
3

Run

Start the dev server. Use Expo Go for OpenRouter only, or build a dev client for on-device AI.

cd my-app && npx expo start npx expo run:ios # or expo run:android
